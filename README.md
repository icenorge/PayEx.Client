## About
`PayEx.Client` is a `netstandard2` library to talk to PayEx direct REST APIs.

## Supported APIs:
- Vipps 
  - create payment
  - Vipps authorize
  - capture 
  - cancel
  - reversal
- CreditCard 
  - create payment
  - create recurring payment with initial payment
  - create recurring payment without initial payment
  - capture
  - cancel
  - reversal
